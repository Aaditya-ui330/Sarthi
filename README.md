Sarthi is a crowdsourced public safety web application that analyzes time-aware
incident data to visualize urban safety through dynamic heatmaps and risk-based
route evaluation.

## Features
- Crowdsourced incident reporting
- Time-decay based risk scoring
- Dynamic safety heatmaps
- Risk-aware route evaluation
- Context-aware insights by time of day

## Tech Stack
- Frontend: HTML, CSS, JavaScript, Leaflet
- Backend: Python (Flask)
- Authentication: Firebase

## Purpose
Built as a mini project to explore spatio-temporal data analysis, user-driven
systems, and safety-focused application design.
